# Customer Shopping Behavior Analysis
End-to-end data analytics project using Python, SQL, and Power BI
## Project Overview
This is an end-to-end data analytics project focused on analyzing customer shopping behavior. The project demonstrates the complete analytics workflow using Python, SQL, and Power BI — from raw data cleaning to business insights and dashboard creation.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL (PostgreSQL)
- Power BI
- Jupyter Notebook

## Dataset
Customer shopping behavior dataset containing demographic details, purchase history, discounts, subscription status, shipping types, and product categories.

## Project Workflow
1. Data Cleaning & Exploration using Python
2. Data Storage & Querying using SQL
3. Business Question Analysis
4. Interactive Dashboard Development in Power BI
5. Insight Documentation & Reporting

## Key Business Questions
- Revenue contribution by gender and age group
- Impact of discounts on customer spending
- Customer segmentation based on purchase history
- Subscription vs non-subscription spending behavior
- Top products and categories by sales and reviews

## Files in This Repository
- `customer_shopping_behavior.csv` – Raw dataset  
- `customer_shopping_behavior.ipynb` – Data cleaning & analysis  
- `customer_behavior_analysis.sql` – SQL business queries  
- `customer_behavior_dashboard.pbix` – Power BI dashboard  
- `Customer_Shopping_Behavior_report.pdf` – Detailed project report  
- `Ppt_customer_behavior_analysis.pptx` – Presentation slides  

## Key Insights
- Young adult customers contribute the highest revenue.
- Loyal customers generate high value despite smaller numbers.
- Discounts increase purchase volume but may reduce profitability.
- Subscribed customers show higher average spending.
- Clothing and accessories dominate total sales and revenue.

## Conclusion
This project showcases practical experience in solving business problems using data and delivering insights through interactive dashboards.
